CLOPS — Command Line OPtionS

CLOPS is…

CLOPS is a Java parser generator for command line options. It is also a documentation generator for command line interfaces. Using CLOPS guarantees that your code and your documentation are in sync.

Example

1. Describe the command line

NAME:: Foo
ARGS::
  SayHi: {"--say-hi", "-sh" }
  SayBye: {"--say-bye", "-sb"}
  Help: {"--help", "-h"}
FORMAT:: (SayHi | SayBye)* | Help;

3. FooBar

NAME:: Foo
ARGS::
  SayHi: {"--say-hi", "-sh" }
  SayBye: {"--say-bye", "-sb"}
  Help: {"--help", "-h"}
FORMAT:: (SayHi | SayBye)* | Help;

2. Use the generated parser

FooParser p = new FooParser();
List errors = p.parse(args);
if (!errors.isEmpty()) {
  report(errors, System.err);
  p.printUsage(System.out);
}
FooOptionsInterface opt = 
    p.getOptionsInterface();
if (opt.isHelpSet())
  p.printUsage(System.out);
if (opt.isSayHiSet()) 
  System.out.println("hi");
if (opt.isSayByeSet())
  System.out.println("bye");
