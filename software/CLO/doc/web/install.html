CLOPS installation

First, make sure that you have Java 6, or newer, installed.

  1. Download the distribution: clops-0.1.zip, clops-0.1.tar.gz, or clops-0.1.tar.bz2.
  2. Extract files from the archive. The place where the directory clops-0.1 lies after this operation will be referred later as DIR.
  3. Set the environment variable CLOPSPATH to point to clops-0.1/lib.
    • Linux. Add a line to ~/.bashrc: 
              export CLOPSPATH=DIR/clops-0.1/lib
    • Windows.
      1. Go to Control Panel, the classic view.
      2. Open System.
      3. Go to Advanced.
      4. Click on Environment variables.
      5. Add the variable CLOPSPATH and give it the value DIR/clops-0.1/lib
    In both cases, remember to replace DIR by the path of the directory where you extracted the files.
